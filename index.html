OLGA: Open Limited-area Gliding Analysis

About

OLGA is a Python framework, built around the WRF-ARW mesoscale model, to produce gliding specific weather forecasts. It automates all the steps involved; downloading the GFS input data, running WRF, post-processing the output of WRF, and creating maps/time series/soundings. OLGA is still in development: we are busy testing the code, validating the setup, etc. Anyone who is interested in using OLGA is welcome to do so - we would be happy to help - but don't expect a finished product :-) When the time is right, an official 1.0 version will be released.
